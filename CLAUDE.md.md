# MissFit Mindset — Brand Identity Guide for Claude

## Who We Are

MissFit Mindset is a coaching brand focused on \[your positioning].
All output — documents, presentations, copy, designs — must reflect this identity.

## Brand Voice

* Tone: Bold, empowering, direct, warm
* Avoid: Corporate jargon, passive voice, generic motivational clichés
* Always: Speak to transformation, mindset shifts, real results

## Color Palette

* Primary: #\[your hex] — used for headings, CTAs, key accents
* Secondary: #\[your hex] — used for backgrounds, supporting elements
* Accent: #\[your hex] — sparingly, for highlights only
* Neutral Dark: #\[your hex]
* Neutral Light: #\[your hex]

## Typography

* Heading Font: \[Font Name] — Bold, use for all H1/H2
* Body Font: \[Font Name] — Regular 400/500 weight
* Never use: Arial, Times New Roman, or system defaults

## Logo Usage Rules

* Logo files are in /assets/logos/
* Always use logo-primary.png on light backgrounds
* Always use logo-dark.png on dark/colored backgrounds
* Minimum size: 120px width. Never stretch or recolor.
* Clear space: Equal to the height of the "M" in MissFit on all sides

## Document Styling Rules

When generating any document (Word, PDF, HTML, slides):

1. Use the document header template at /assets/templates/document-header.html
2. Apply the primary color to all H1 headings
3. Apply the secondary color to section dividers
4. Footer must always include: MissFit Mindset logo + website URL + tagline
5. Page margins: 1 inch all sides (or 25mm)
6. Never use default Word/Google Doc styling

## File Naming Convention

\[project]-\[type]-\[version]-\[date].extension
Example: onboarding-proposal-v1-2026-06.docx

## What Claude Should Always Do

* Before generating any branded document, confirm the template and colors match this guide
* When in doubt about a design choice, default to the primary color and heading font
* Reference /assets/templates/ for any document structure
